# kick: git sync automator

# ABOUT

`kick` automates a common git workflow:

1. Stage all local changes.
2. Commit all local changes.
3. Pull remote changes.
4. Push local changes.

# EXAMPLE

```console
$ kick
```

See `kick -help` for more options.
